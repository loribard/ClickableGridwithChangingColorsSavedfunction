AltSchool Challenge

Create a board of n x m tiles on a web page. Each tile on the board will start as white, but when clicked will toggle between a generated random color and returning to white. Each time a tile is toggled to a random color it should generate a new random color value. Include a button so that users can clear (set all tiles to white) on their board. And additionally include a save button that will persist the state of the board to a server side datastore. Instructions: * Respond with your attached completed solution. * Use any languages, frameworks, or 3rd party libraries as you see fit. * Let us know if you have any questions as you're attempting the challenge.
